# Lab 13 – Infrastructure as Code using AWS CloudFormation

## Objective
To automate cloud infrastructure deployment using AWS CloudFormation by creating, updating, and deleting resources through a YAML template.

---

## Tasks Performed
- Created a CloudFormation YAML template
- Defined an EC2 instance inside the template
- Launched a CloudFormation stack using the template
- Verified EC2 instance creation from the stack
- Updated the template to change instance type
- Applied stack update to reflect changes
- Verified updated EC2 configuration
- Deleted the CloudFormation stack
- Confirmed automatic termination of resources

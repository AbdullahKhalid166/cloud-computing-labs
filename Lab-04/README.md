To deploy and manage a web application using AWS Elastic Beanstalk.

---

---

### 2. Access Application

- Checked environment dashboard
- Monitored health status

### 4. View Metrics
- Observed performance metrics of application

---

### 5. Configure Auto Scaling
- Enabled auto scaling for application

---

### 6. Deploy New Version
- Updated application and redeployed new version

---

Successfully deployed and managed a scalable web application using AWS Elastic Beanstalk.

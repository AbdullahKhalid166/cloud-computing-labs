# Lab 06 – Database Management using Amazon RDS

## Objective
To create and manage a cloud database using Amazon RDS, including data insertion, backup, and high availability features.

---

## Tasks Performed
- Created a database instance using Amazon RDS
- Configured security group to allow database access
- Connected to the database and created tables
- Inserted multiple records into the database
- Took a database snapshot for backup
- Restored the database from snapshot
- Verified restored data
- Studied Multi-AZ deployment for high availability

---

## Key Implementation
- Database created and accessed successfully
- Records were inserted and retrieved
- Snapshot was taken and restore process completed
- Data remained consistent after restoration
- Multi-AZ concept understood (failover and replication)

---

## Result
Successfully deployed and managed a cloud-based relational database with backup and recovery features.

---

## Learning Outcomes
- Hands-on experience with Amazon RDS
- Database creation and data insertion
- Backup and restore using snapshots
- Understanding high availability with Multi-AZ

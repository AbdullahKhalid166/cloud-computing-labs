# Lab 03 – Serverless Computing using AWS Lambda

## Objective
To understand serverless computing by creating and managing AWS Lambda functions and integrating them with S3.

---

## Tasks Performed

### 1. Create Lambda Function
- Created a Lambda function using Python
- Returned a simple response:
  "Hello from Lambda"

---

### 2. Create S3 Bucket
- Created an S3 bucket for trigger-based operations

---

### 3. Configure S3 Trigger
- Linked S3 bucket with Lambda function
- Triggered Lambda automatically on file upload

---

### 4. Monitor Using CloudWatch
- Checked Lambda execution logs using CloudWatch
- Verified successful execution

---

### 5. Modify Lambda Function
- Updated function output to:
  "Successfully Triggered!"

---

### 6. Test File Upload
- Uploaded file to S3 bucket
- Verified automatic Lambda execution

---

### 7. Delete Objects
- Removed uploaded files from S3 bucket

---

## Result
Successfully implemented a serverless architecture using AWS Lambda and S3 triggers.

---

## Learning Outcomes
- Understanding of serverless computing
- Lambda function creation and execution
- S3 event triggers
- CloudWatch monitoring
